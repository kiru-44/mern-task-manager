# MERN Task Manager

A full-stack **Task Manager Web Application** built using the MERN Stack (MongoDB, Express.js, React.js, Node.js).  
Users can register, login, and manage their personal tasks efficiently.

---

## Features

- User Registration & Login
- JWT Authentication
- Create Tasks
- Mark Tasks as Completed
- Delete Tasks
- Protected Routes
- Responsive Dashboard UI

---

## Tech Stack

Frontend:
- React.js
- React Router
- Axios
- CSS

Backend:
- Node.js
- Express.js

Database:
- MongoDB
- Mongoose

Authentication:
- JSON Web Token (JWT)
- bcrypt
